# Data Portfolio

Portfolio with practical projects in SQL, SQL Server, Power BI, Power Query, C# and Python.

## Projects

### 1. Dashboard Power BI – Analiză vânzări
Power BI dashboard for sales analysis, using KPI indicators, filters and interactive visuals.

### 2. Dashboard Power BI – Analiză retail
Multi-page Power BI dashboard for retail analysis, including overview, products, clients and stores.

### 3. Proiect Power BI + Power Query – Data Cleaning și Dashboard de vânzări
Project focused on data cleaning, validation, transformation and dashboard creation using Power Query and Power BI.

### 4. Retail Sales Database – SQL Server
Relational database project for retail sales analysis, including tables, relationships, SQL queries, view and stored procedure.

### 5. RetailSalesViewer – C#
Windows Forms application connected to SQL Server, used to display data from a SQL view in a DataGridView.

### 6. Game2048 – Python
Simple Python project that implements the logic of the 2048 game.
